# Brinstart 🎮🕹️🎯

Brinstart is a Planning Poker like you've never seen before.
This repository is responsible for everything beautiful that you see in our application. Enjoy 💅🏽

### Setup

**Setting the node version**
```bash
nvm use
```
**Install dependencies**
```bash
npm install
```
**Run the project**
```bash
npm run start
```
